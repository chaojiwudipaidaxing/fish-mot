# Paper Draft Workspace

This directory stores a lightweight, engineering-style paper scaffold.

- Main skeleton: `paper/paper_skeleton.md`
- Auto-generated result narrative: `results/paper_assets_val/results_section.md`
- Auto-generated tables/figures:
  - `results/paper_assets_val/paper_main_table.tex`
  - `results/paper_assets_val/paper_count_table.tex`
  - `results/paper_assets_val/main_table_val_seedmean_std_paper.png`
  - `results/paper_assets_val/count_stability_bar_paper.png`
  - `results/paper_assets_val/stratified_metrics_val.png`
  - `results/paper_assets_val/runtime_profile.png`

Use the skeleton as a starting point, then replace smoke-test numbers with full-run results.
